Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd Breytir working directory 
	cd ~ Breytir working directory í "Home"
	ls Skrifar öll skjöl sem eru í working directory
	pwd Skrifar working directory
	mkdir Býr til directory eða folder


2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone Afritar skjal
	git log Skrifar það sem búið er að gera í git repository
	git status Skrifar hvernig staðan er á repository
	git diff Skrifar hvað mismunurinn er á tveimur greinum í git
	git checkout Skiptir á milli git greinum

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff Sýnir alla mismuni á tveimur greinum
	b) git diff --staged Sýnir mismunin á síðasta commit og núverandi stöðu 
	c) git diff commit1 commit2 sýnir mismunin á tveimur ákveðnum commit 

4. 	Hvað er útgáfustýring (e.version control)?: Það geymir öll fyrrirverandi skjöl sem þú hefur búið til eða breytt þannig að hægt er að ná í þau seinna

5.	Hverjir eru helstu kostir við að nota GIT?: Það að hægt er að komast í gömul og óbreytt skjöl sem annars væri ekki hægt að ná í
